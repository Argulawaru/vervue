# 👗 VERVUE – Outfit Recommendation System

## 📌 Overview

VERVUE is a smart outfit recommendation web application that helps users organize their wardrobe and generate outfit combinations. Users can upload their clothing items and receive personalized outfit suggestions for daily or weekly use.

---

## 🚀 Features

* 📤 Upload clothing items (shirts, pants, etc.)
* 👀 View and manage your wardrobe
* 🎯 Generate outfit combinations
* 🧪 Try-on / preview outfits
* 📅 Weekly outfit planning
* 🧠 Scalable for AI-based recommendations

---

## 🛠️ Tech Stack

* Frontend: React.js
* Backend: Firebase
* Database: Firestore
* Styling: CSS

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/Argulawaru/vervue.git
```

2. Navigate to the project folder:

```bash
cd vervue
```

3. Install dependencies:

```bash
npm install
```

4. Run the app:

```bash
npm start
